## Название проекта
Разработка Telegram-бота для защиты документов с использованием цифровых водяных знаков

# Краткое описание
Telegram-бот, который позволяет защищать документы (PDF, DOCX, изображения) с помощью водяных знаков. Бот реализует нанесение видимых и скрытых меток, проверку подлинности дркументов и контроль целостности файлов через SHA-256 хэширование.

## Требования 
- Python 3.9
- pip
- Telegram аккаунт для тестирования бота
