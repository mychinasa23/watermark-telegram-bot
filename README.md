# Название проекта
Разработка Telegram-бота для защиты документов с использованием цифровых водяных знаков

Программа реализует Telegram-бота, который позволяет защищать документы (PDF, DOCX, изображения) с помощью цифровых водяных знаков. Бот поддерживает нанесение видимых и скрытых меток, проверку подлинности документов и контроль целостности файлов через SHA-256 хэширование.

# Как запустить

### 1. Скачайте проект 

```bash
https://github.com/mychinasa23/watermark-telegram-bot.git
cd watermark-telegram-bot
```

### 2. Создаёте виртуальное окружение

```bash
# Windows
python -m venv .venv
.venv\Scripts\activate

# Linux / macOS
python3 -m venv .venv
source .venv/bin/activate
```

### 3. Установите зависимости

```bash
pip install -r requirements.txt
```

### 4. Создайте файл .env с токеном бота
Создайте в корне проекта файл .env и напишите в нём:

```text
BOT_TOKEN=ваш_токен_от_BotFather
DB_PATH=bot_database.db
TEMP_DIR=temp_files
```

- Как получить токен: Напишите @BotFather в Telegram → /newbot → укажите имя бота → скопируйте токен

### 5. Запустите бота

```bash
python bot.py
```

### 6. Проверьте работу

- Откройте Telegram
- Найдите вашего бота по имени
- Нажмите "старт"
