# Название проекта
Разработка Telegram-бота для защиты документов с использованием цифровых водяных знаков

Программа реализует Telegram-бота, который позволяет защищать документы (PDF, DOCX, изображения) с помощью цифровых водяных знаков. Бот поддерживает нанесение видимых и скрытых меток, проверку подлинности документов и контроль целостности файлов через SHA-256 хэширование.
